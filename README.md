## ✨ Features
- **100% On-Device:** Uses MediaPipe Hands running entirely in your browser. No video, images, or telemetry are ever sent to a server.
- **Hands-Free Scrolling:** Move your hand up to go to the previous video, and down to go to the next.
- **Configurable:** Adjust sensitivity, cooldown, motion thresholds, and handedness in the options page.
- **Platform Agnostic Architecture:** Designed with a platform adapter pattern to easily support new video feeds (e.g. TikTok) in the future.

## 🚀 Installation (Developer Mode)

1. Clone or download this repository.
2. Build the extension:
   ```bash
   npm install
   npm run build
   ```
3. Open Chrome and navigate to `chrome://extensions`.
4. Enable **Developer mode** in the top right corner.
5. Click **Load unpacked** and select the `dist` folder generated by the build process.

## 🎮 How to Use

1. Navigate to a supported platform:
   - **YouTube Shorts:** `youtube.com/shorts/*`
   - **Instagram Reels:** `instagram.com/reels/*`
2. Grant camera permissions when prompted.
3. Keep your hand visible in your webcam frame. 
4. Move your hand **down** to scroll to the next video.
5. Move your hand **up** to scroll to the previous video.
6. Click the extension icon in the toolbar to see real-time pipeline status, confidence, and FPS, or to turn the extension off.

## 🛠️ Development

- `npm run watch`: Starts the build process in watch mode, automatically recompiling changes to the `src` directory.

---

## 🔒 Privacy Guarantee

GestureScroll does not connect to any external cloud services for inference. All machine learning models (`.wasm` / `.data`) are bundled directly with the extension and run locally within the extension's isolated sandbox.
